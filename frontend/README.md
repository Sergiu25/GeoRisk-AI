# GeoRisk AI Frontend

## Framework: Angular

This folder will contain the Angular-based frontend application for GeoRisk AI.

### Technology Stack
- **Framework**: Angular 18+
- **Package Manager**: npm
- **Build Tool**: Angular CLI
- **Styling**: Tailwind CSS or Bootstrap

### Planned Structure
```
frontend/
├── src/
│   ├── app/
│   │   ├── components/
│   │   ├── services/
│   │   ├── models/
│   │   └── pages/
│   ├── assets/
│   └── styles/
├── angular.json
├── package.json
└── tsconfig.json
```

### Next Steps
1. Initialize Angular project with `ng new`
2. Set up HTTP client for backend API communication
3. Create main components (location management, risk assessment display)
4. Integrate Swagger API documentation
